# LiveCodeShare

A real-time collaborative code editor built with **JavaFX** and **Firebase**. This tool allows developers to share and edit code snippets simultaneously, providing a synchronized environment for pair programming and remote collaboration.

## 📋 Prerequisites

*   **Java Development Kit (JDK):** 17 or higher.
*   **Maven:** Installed and configured in your system path.
*   **Firebase Account:** A project set up on the [Firebase Console](https://console.firebase.google.com/).

## ⚙️ Setup & Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/MdMahbubTanmay/LiveCodeShare.git](https://github.com/MdMahbubTanmay/LiveCodeShare.git)
    cd LiveCodeShare
    ```

2.  **Firebase Configuration:**
    *   Download your `serviceAccountKey.json` from the Firebase Console (Project Settings > Service Accounts).
    *   Place the file in `src/main/resources/` or the designated config folder.
    *   Update your database URL in the source code where the Firebase options are initialized.

3.  **Build the Project:**
    ```bash
    mvn clean install
    ```

4.  **Run the Application:**
    ```bash
    mvn javafx:run
    ```

## 🖥️ Usage

1.  Launch the application.
2.  Enter a **Room ID** or create a new session.
3.  Share the ID with a colleague.
4.  Start coding! Changes made by any participant will reflect instantly for everyone else.
